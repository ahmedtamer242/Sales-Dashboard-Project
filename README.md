# 📊 Sales Dashboard Analysis

This is a comprehensive sales dashboard project developed using Microsoft Excel. The goal of this project is to transform raw data into actionable insights to help decision-makers understand company performance.

## 🚀 Project Overview
This project analyzes sales data to identify key patterns and trends. It includes an interactive dashboard that allows users to explore data based on categories, regions, and years.

## 📈 Key Features
* **Sales Analysis:** Displays total sales, profit, and discount figures.
* **Classification:** Analyzes sales by category and sub-category.
* **Geographical Analysis:** Identifies the top 10 cities and states by sales volume.
* **Temporal Evolution:** Tracks sales growth and order counts across years (2014-2017).

## 🛠 Tools Used
* **Microsoft Excel:** (Pivot Tables, Charts, Slicers, Dashboard Design).

## 📊 Key Findings
Through this analysis, I extracted key performance indicators (KPIs) including:
* Total realized sales.
* Net profit.
* Distribution of orders by region.
