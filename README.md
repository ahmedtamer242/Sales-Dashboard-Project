📊 Sales Dashboard AnalysisThis is a comprehensive sales dashboard project developed using Microsoft Excel. The goal of this project is to transform raw data into actionable insights to help decision-makers understand company performance.  🚀 Project OverviewThis project analyzes sales data to identify key patterns and trends. It includes an interactive dashboard that allows users to explore data based on categories, regions, and years.  📈 Key FeaturesSales Analysis: Displays total sales, profit, and discount figures.  Classification: Analyzes sales by category and sub-category.  Geographical Analysis: Identifies the top 10 cities and states by sales volume.  Temporal Evolution: Tracks sales growth and order counts across years (2014-2017).  🛠 Tools UsedMicrosoft Excel: (Pivot Tables, Charts, Slicers, Dashboard Design).  📊 Key FindingsThrough this analysis, I extracted key performance indicators (KPIs) including:Total realized sales.  Net profit.  Distribution of orders by region.  📂 File StructureSales_Dashboard.pdf: Preview version of the final dashboard.
